ctrl+alt+T打开终端
ls           显示当前目录下的文件
ls -a       显示隐藏的文件
vi           修改文件内容
rm         删除文件内容
sudo apt-get install +软件名  安装软件
touch +想要创建的文件名        
cp -i file1(old) filecopy(new) 文件复制到文件（interact）
cp file folder/ 将文件复制到文件加
cp -R file folder/ 将文件夹递归到另一个文件夹下面（recursive循环的）循环复制
file*   指的是与file开头的所有文件
mv file folder  将一个文件移动到另一个文件夹
mkdir folder1（/f1） 创建文件夹
rmdir folder 删除文件夹（只能删除空的文件）
rm -r folder/ 可以直接删除文件夹
nano             相当于一个文本编译器
cat                 查看文件内容
cat  file >file2将file复制到file1
cat file1 file2 > file3 两个数据一起复制
cat file1 >> file2 添加而不是替代
